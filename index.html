Basic Shortcuts

VSCode provides several basic shortcuts to enhance your productivity.

Common actions include opening files, saving, and undo/redo.

Ctrl + P - Quick Open File
Ctrl + S - Save File
Ctrl + Z - Undo

These are essential for daily use.

  • Open file: Ctrl + P
  • Save file: Ctrl + S
  • Undo: Ctrl + Z
  • Redo: Ctrl + Y
  • Close file: Ctrl + W
Navigation Shortcuts

Quick navigation within VSCode is essential for productivity.

Use shortcuts to move between files and sections.

Ctrl + Tab - Switch tabs
Ctrl + Shift + E - Open Explorer
Ctrl + B - Toggle Sidebar
  • Switch tabs: Ctrl + Tab
  • Open Explorer: Ctrl + Shift + E
  • Toggle Sidebar: Ctrl + B
  • Go to next editor group: Ctrl + 1, 2, 3...
  • Zoom in/out: Ctrl + = / Ctrl + -
Editing Shortcuts

Editing is faster with these shortcuts.

They help with selecting, duplicating, and deleting lines efficiently.

Ctrl + Space - IntelliSense
Ctrl + D - Select next instance
Ctrl + / - Comment line
  • Trigger IntelliSense: Ctrl + Space
  • Duplicate line: Shift + Alt + Down Arrow
  • Delete line: Ctrl + Shift + K
  • Move line up/down: Alt + Up/Down Arrow
  • Find/Replace: Ctrl + F
Debugging Shortcuts

Debugging is easier with these shortcuts.

They allow starting, stopping, and controlling debugging sessions.

F5 - Start debugging
F9 - Toggle breakpoint
Ctrl + Shift + D - Debug Panel
  • Start debugging: F5
  • Toggle breakpoint: F9
  • Debug Panel: Ctrl + Shift + D
  • Step into: F11
  • Step out: Shift + F11
Customization Shortcuts

Customize VSCode using these shortcuts.

They allow adding extensions and changing settings quickly.

Ctrl + Shift + P - Command Palette
Ctrl + Shift + X - Extensions View
Ctrl + , - Settings
  • Command Palette: Ctrl + Shift + P
  • Extensions View: Ctrl + Shift + X
  • Settings: Ctrl + ,
  • User Snippets: Ctrl + Shift + S
  • Toggle Zen Mode: Ctrl + K, Z
